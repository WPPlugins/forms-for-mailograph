Forms for Mailograph will enable Wordpress to use Mailograph email marketing features.  

== Description ==

Forms for Mailograph enables you to integrate your Wordpress website with Mailograph email marketing. It will make the email marketing easy! Just include the widget in any area you want and you have a registration form connected directly to your Mailograph account. 

= What can I do with Mailograph? = 

* You can create a list and grow the list by putting the subscription form directly in your Wordpress website. 
* You can set Autoresponders in your Mailograph account, send scheduled emails to subscriber upon their subscription to list.
* You can send Email Campaigns to subscribers on scheduled time.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/forms-for-mailograph` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Appearence-> Widget 
4. You can select Forms for Mailograph and add it to a any area you want.
5. Enter your API Key, List Id, etc to setup your form
6. Sit back and enjoy!



== Frequently Asked Questions ==

= Do I need to sign up to Mailograph to be able to use Forms for Mailograph? =

Yes. You need to have an account in Mailograph. 

= Can I have many forms linked to different lists in Mailograph =

Yes. You can have unlimited forms linked to unlimited lists.
